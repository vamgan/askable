How it works

Three steps. Done.

1

Annotate

Add data-askable to any element. The attribute holds metadata for your AI — the div itself can contain any content: charts, tables, text, images, whatever.

2

Observe

Call ctx.observe(document) once. askable attaches listeners to every annotated element — click, focus, hover. MutationObserver handles dynamic content.

3

Inject

At your AI boundary, call ctx.toPromptContext(). Drop the string into your system prompt. The model knows exactly what the user is looking at.
