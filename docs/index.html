How it works

Three steps. Done.

1

Annotate

Add data-askable to any element with a JSON payload — metric name, value, whatever context your AI needs.

2

Observe

Call ctx.observe(document) once. askable attaches listeners to every annotated element — click, focus, hover.

3

Inject

At your AI boundary, call ctx.toPromptContext(). Drop it into your system prompt. The model knows exactly what the user sees.
